--- 구현 완료 (2026-02-27) ---
설계 대비 미구현 7건 구현 완료:
1. 실수 한도 토글 - mistakeLimit OFF 시 게임오버 체크 스킵, 실수 표시 숨김
2. 스마트 힌트 토글 - smartHints OFF 시 직접 공개(findDirectReveal)만 사용
3. 통계 메시지 토글 - statsMessage OFF 시 완료 화면 메시지 비표시
4. 자동 잠금 방지 - Wake Lock API로 게임 중 화면 꺼짐 방지
5. 최고 점수 기간별 리셋 - 날짜/주/월 변경 시 today/thisWeek/thisMonth 자동 초기화
6. 규칙 화면 - 프로필 메뉴에서 규칙 화면 진입, 3가지 규칙 정적 콘텐츠
7. 숫자 우선 입력 모드 - numberFirst ON 시 숫자 잠금→셀 탭→자동 입력
